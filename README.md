# rdsbridge-site

The landing page for **RDS Bridge**, served at [rdsbridge.com](https://rdsbridge.com) via GitHub Pages.

This repo is the website only. The application itself lives at
**[m0euk/RDS-Bridge](https://github.com/m0euk/RDS-Bridge)** — that's where the code,
the releases and the documentation are.

Static and hand-written: one `index.html` with inline CSS, no JavaScript, no build step.

MIT licensed, same as the application.
